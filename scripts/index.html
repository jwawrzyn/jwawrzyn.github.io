Raspberry Pi Wireless Photo Frame

I got the idea for this from http://www.ofbrooklyn.com/2014/01/2/building-photo-frame-raspberry-pi-motion-detector/. I thought that it would be really awesome and a great way to get into a Raspberry Pi project and I love showing off my photos. I took what he did and got my raspberry pi up and running, tested out a sensor and used some of his code and modified some to deal with flickr authentication due to the fact that my photos are mostly private.

I also just updated it to use play nicer with flickr's update to SSL.

You can check out the code at: https://github.com/jwawrzyn/raspberrypiframe
